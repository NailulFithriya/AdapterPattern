Topik : Note

Desain Pattern yang dipakai adalah Strategy Pattern dan Adapter Pattern sesuai instruksi soal.

Strategi pattern dipakai untuk memilih tools mana yang ingin dipakai untuk menulis catatan.
tersedia 2 tools yang ditawarkan yaitu menggunakan OneNote dan Notepad. pengguna dapat memilih salah satu
tools tersebut. Apabila pengguna memilih OneNote otomatis akan diarahkan untuk membuka OneNote, begitu pula
jika pengguna memilih opsi notepad, maka tools yang digunakan untuk menulis catatan adalah notepad.

Adapter Pattern digunakan untuk mengganti tema pada aplikasi catatan yang digunakan. disini saya membuat
tema berganti menjadi dark secara default. apabila pengguna ingin mengganti tema maka otomatis tema akan berganti
menjadi warna dark.
